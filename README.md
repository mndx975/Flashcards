# Программа: Быстрые карточки для повторения



**Краткое описание:** Программа позволяет быстро повторить материал с помощью карточек.

##  Как запустить из исходного кода

1.  Убедитесь, что у вас установлен Python 3.8 или выше. 
    
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
3. Запустите основной скрипт:
    ```bash
    python main.py
    ```

##  Готовая сборка (для Windows)

Если вы не хотите разбираться с Python, вы можете запустить готовую программу:
1. Запустите .exe файл:
    ```bash
    main.exe
    ```

**Внимание:** Антивирус может ругаться на подобные файлы. Это ложное срабатывание.


##  Технологии
*   Python
*   Библиотека tkinter
